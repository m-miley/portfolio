NASS Honey Bees

Using Tableau and 3 NASS csv files containing data spanning six years on the United States honeybee population, health, and productivity, I completed a project highlighting some concerns about health and the impact of colony collapse disorder. I created 2 dashboards displaying insights and connected the filters to maximize interactivity for the viewer to further explore the data. Finally, I presented the data to a group of professionals using Story Mode in Tableau from a remote setting. Click the Tableau Public button below to see the published workbook.

Baywheels San Francisco

This project utilizes postgreSQL on pgAdmin4 to analyze Lyft bikeshare data in San Francisco, CA. I wrote and executed SQL queries to investigate the relational database to reveal insights that confidently suggest maintenance timing, distribution schedules, and transportation updates. Visualizations created in Excel and presented through PowerPoint amongst a group of peers in a 10 minute presentation.

Mission to Mars

Bootcamp project using Beautiful Soup and Splinter to scrape data from multiple websites, store it in MongoDB, and rebuild to display the results in a Flask app local web page. The page has a button built in at the header to activate web scraping scripts to collect the most recent data. Finally, the data is loaded to the page from an html file.

World Weather Analysis

PlanMyTrip is a project that employs API interactions through multiple scripts to provide real-time suggestions to users for destinations based on ideal weather preferences. By collecting and analyzing global data and monitoring weather parameters, this project offers visualizations with which the user can discover and make travel decisions such as destination, lodging, current weather, and trip itinerary. Python, Pandas, Matplotlib, Google APIs and gmaps.
